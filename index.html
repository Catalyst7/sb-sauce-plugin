Sauce for Se Builder


Se Builder is a Firefox addon for creating Selenium tests. Sauce for Se Builder is a plugin for it that adds support for Sauce Labs' infrastructure.

Install Se Builder Now »

Features

  • Play back Selenium 2 scripts on Sauce OnDemand.
  • Export scripts ready for OnDemand playback.

Installation

To get Sauce for Se Builder, first install the newest version of Se Builder, then install the plugin from the plugins manager in Builder.
